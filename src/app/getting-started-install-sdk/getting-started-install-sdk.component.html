Getting Started - Install the SDK

Get Unity3D Setup to use the SideQuest Legends SDK

Step 1 - Install Dependencies
Create a fresh project ( Unity version 2019.2.X recommended ) and install some extra packages using the unity package manager. You will need to install:

XR Legacy Input Handlers
TextMesh Pro
Unity UI
PostProcessing Stack ( not required yet but soon hopefully )

You may already have some of these installed and can skip them. See the first picture above for instructions.


Step 2 - Enable VR in unity ( Not required by handy to get out of the way if you want to test in VR later )
Open the player settings in unity and enable virtual reality under XR Settings. See the second image above for instructions.


Step 2 - Download & Install
Download the Legends SDK unity package from here. Drag the file over the Project panel in Unity to install it into your project.
