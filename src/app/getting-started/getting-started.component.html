Getting Started - Build an App

Use the SDK to build a squanch app

Step 1 - Open
Open the SideQuest option in the Tools menu of Unity3D and select a GameObject in your hierarchy panel.


Step 2 - Add Scripts
Click Next to scan the selected game object for AppBehaviours or custom scripts. Note: The defaults here are fine in most cases. If you are unsure just click Next. This will scan the selected GameObject for an AppBehaviour script and attempt to find any dependencies for it automatically. The pre-selected items here have been detected as dependencies, but you may also need to select more scripts if their types are not defined in the AppBehaviour anywhere.


Step 3 - Build Options & Run Build
Select Quick Build to skip the mobile build variant - this greatly reduces build times during testing and allows you to test the app on desktop ( or mobile with Oculus Link ). You only need to do a complete a non quick build when you are ready to publish your app. Select Build SideQuest App to start the build process. Keep an eye on the console tab for any issues.



Step 4 - Grab Your Squanch Files!
Your app will be compiled into the output folder. If you selected quick build there will be one file otherwise there will be two, gameobject.squanch and gameobject.squanch_android. These files can be loaded directly into SideQuest Legends by creating a space here or you can run them in the SideQuest Inspector to test.
