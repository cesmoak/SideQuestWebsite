Getting Started - Play test with the SideQuest Inspector

Run your SideQuest Legends App inside Unity3D.

Back To Developers




Step 1 - Open
Open the SideQuest option in the Tools menu of Unity3D as seen in the image above.


Step 2 - Enter a Space ID
Enter a Space ID to launch SideQuest Legends in a specific space - this is good for testing against other apps created with SideQuest Legends.


Step 3 - Run Your Squanch File
Select your SideQuest app ( .squanch ) and run it inside the Unity3D Editor. It will run the full SideQuest Legends Game so you will see the loading screens and then it will load into your selected space. You can test all sorts of things in this way like interaction, networking, mult-iuser, interoperability with other SideQuest Legends apps. You have full access to the Unity3D hierarchy, inspector and console panels to be able to test, prototype and tweak your apps as you build.
