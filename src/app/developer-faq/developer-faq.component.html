SideQuest Inspector

- What is the SideQuest Inspector and why do I need it?
When we started building the SideQuest Legends SDK we realised that to provide developers with powerful scripting capability was great and all but pretty soon it was clear we had to do more in order to provide a complete developer solution. We needed to find a way to give developers as much power as us when we build SideQuest Legends. To solve this problem we built the SideQuest Inspector.


- What benefit does the SideQuest Inspector offer?
The SideQuest Inspector is the most powerful tool in the SideQuest Legends SDK. The beautiful part about the SideQuest Inspector is that it allows you to run the full SideQuest Legends game platform inside the Unity3D editor with full access to the Unity3D tools for inspecting, debugging and improving performance. This means we can allow developers to have as much power as we do to build the best apps they can on SideQuest Legends.


- How do I get the SideQuest Inspector and set it up?
The SideQuest Inspector is included with the SideQuest SDK, it automatically gets downloaded the first time you import the SDK. The SIdeQuest Inspector must run in an empty Unity3D ( 2019.2.9f ) project which is also provided as part of the SideQuest SDK. If you open the SideQuest SDK as ususal and click the "Run SideQuest Inspector" button it will open an empty Unity3D project and import the SIdeQuest Inspector automatically.

Alternatively you can download the SideQuest Inspector Unity3D packge from here and drag and drop it into an empty Unity3D project window.
