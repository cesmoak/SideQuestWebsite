SideQuest Scripting - Interactables

Get going with scripting in SideQuest to open a whole world of possibilities to create content.

NOTICE This next section assumes you are familiar with Unity3D and C-Sharp scripting. Unity has a lot of great tutorials to get your started.

WARNING Try to avoid adding multiple nested RigidBodies or other Colliders that are not triggers inside the interactables, in some cases it can make them go haywire.

Legends uses the Unity XR Interaction Toolkit ( still in preview ) to provide a native way to interact with objects in VR. It uses interactors ( hands, fixed attach points ) and interactables ( objects you can pick up, throw ) to create a natural way of working with objects in a space. We have improved upon the standard components included in the interaction toolkit to add support for automatic network sync and spawner capabilities. Use the Interactable component from within the Unity Editor to make any GameObjects interactable.

Here is a list of the available options on the Interactable component:

Attach Transform Attach point to use on this Interactable (will use RigidBody center if none set).

Attach Ease In Time Time it takes to ease in the attach (time of 0.0 indicates no easing).

Track Position Whether the this interactable should track the position of the interactor.

Smooth Position Apply smoothing to the follow position of the object.

Smooth Position Amount Smoothing applied to the object's position when following.

Tighten Position Reduces the maximum follow position difference when using smoothing.

Track Rotation Whether the this interactable should track the rotation of the interactor.

Smooth Rotation Apply smoothing to the follow rotation of the object.

Smooth Rotation Amount Smoothing multiple applied to the object's rotation when following.

Tighten Rotation Reduces the maximum follow rotation difference when using smoothing.

Unique ID This reference uniquely identifies this object against other objects in the space ( the default is usually fine ).

Spawner This object will produce another object when selected.

Synced This object is automatically synced to other players.

Respawn Time Time to respawn after being picked up.

Despawn Timeout Time to despawn after being dropped - 0 for never.

Throw On Detach Object inherits the interactor's velocity when released.

Throw Velocity Scale Scale the velocity used when throwing.

Throw Angular Velocity Scale Scale the angular velocity used when throwing.

Gravity On Detach Object has gravity when released (will still use pre-grab value if this is false).

Kinematic On Detach Object is kinematic ( fixed ) when released (will ignore pre-grab value if this is false).

Interactable Events
You can also register listeners on the interactable component to handle the object being picked up / dropped etc. Please note these will only work with the local user and objects they interact with.
