Scripting - Interaction System

Get going with scripting in SideQuest to open a whole world of possibilities to create content.

Notice: This next section assumes you are familiar with Unity3D and C-Sharp scripting. Unity has a lot of great tutorials to get your started.

Adding Interaction to GameObjects
In order to leverage the Legends Interaction System you can call this simple method to enable it. This allows you make a gameobject interactable for users to pick up and throw around. You can also hook into the object with some extra lifecycle methods in your scripts to add more functionality.

WARNING: Try to avoid adding multiple nested RigidBodies or other Colliders that are not triggers inside the interactables, it makes them go haywire. The Legends SDK will automatically add/remove Colliders and RigidBodies to the parent object and manage the properties of both as part of the lifecycle of the interaction system. You can add your own collider / RigidBodies and Legends will use those too but it will stil change properties on them.
